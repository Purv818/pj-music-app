# PJ Music App

A full-stack offline music player with synchronized listening, friend system, and listening rooms — packaged as an Android APK via Capacitor.

## Features

- **Offline Music Player** — scan & play local audio files, queue, shuffle, repeat
- **User Authentication** — email/password with JWT refresh tokens, bcrypt hashing, account lockout
- **Playlists & Favorites** — create playlists, add/reorder songs, favorite tracks
- **Friend System** — send/accept/reject requests, remove friends
- **Synchronized Listening** — peer-to-peer sync invites or group listening rooms
- **Listening Rooms** — owner-controlled rooms, real-time playback sync via Socket.IO
- **Dark & Light Mode** — fully themed with CSS variables
- **Security** — HTTPS/JWT, rate limiting, XSS/CSRF/HPP protection, input validation, bcrypt

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, Vanilla JavaScript |
| Android | Capacitor 5 |
| Backend | Node.js + Express 4 |
| Database | MongoDB + Mongoose |
| Real-time | Socket.IO 4 |
| Auth | JWT (access + refresh tokens) |

## Quick Start

### Backend
```bash
cd pj-music-app/backend
npm install
cp .env.example .env    # edit with your MongoDB URI, JWT secrets, email config
npm run dev             # starts on http://localhost:5000
```

### Frontend → Android APK
```bash
cd pj-music-app/frontend
npm install
# Edit src/js/config.js — set API_BASE_URL and SOCKET_URL to your backend
npx cap add android
npx cap sync android
npx cap open android    # opens Android Studio → Build > Build APK
```
